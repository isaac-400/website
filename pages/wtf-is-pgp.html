PGP

Pretty Good Privacy or PGP is a system to help solve two problems in the digital world:

  1. Privacy → making sure data is only read by the intended recipients
  2. Authenticity → making sure data comes from who it say it comes from
PGP helps with these by providing users with keys that can encrypt and digitally sign data. Each user generates a public and private key, the public one is make public and the private one is kept secret by the owner. Signatures made with the private key can be verified with the public key. Encryption created with the public key can only be unlocked with the private key.

For example, if you sent me an email encrypted with my public key, only I would be able to read it. I sent you a response signed with my private key, you could then verify it came from me with my public key.

Why this matters

Privacy is a fundamental right and authenticity is very valuble. Even if you have nothing to hide, there should be an expectation that no one should read your data without permission. If you feel the same way as I do, try out PGP! This is a good starting resource.
